# Side Projects

A collection of personal projects I've built outside of work.

## [yp](https://yp.elvisbrevi.cl/)

AI-Powered Git Commit and Azure DevOps Management Tool. A .NET global tool that analyzes staged changes and generates professional commit messages automatically, manages Azure DevOps tasks, and supports multiple AI providers (OpenAI, Anthropic, Google Gemini, DeepSeek, OpenRouter).

## [Screenshot Saturday](https://ss.elvisbrevi.cl/)

A curated platform aggregating the best indie game screenshots from Reddit and Bluesky. Built for the gamedev community to discover and share creative work.

## [InfoCandidato](https://infocandidato.elvisbrevi.cl/)

A web application for exploring Chilean electoral data, providing detailed information about candidates and electoral districts for the 2025 elections.

## [Game Prototypes](https://elvisbrevi.itch.io/)

A collection of game prototypes and experiments created using various game engines. Explore experimental gameplay mechanics and creative coding projects.
